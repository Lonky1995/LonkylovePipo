    文件读取的方法：
    

 1. read
read每次读取整个文件，将文件内容放到一个字符串。  
这种方法占内存。  
fread = fhand.read()
 2. 
readline每次读取一行  
这种方法不占内存  
fread = fhand.readline()

 3. 
readlines一次读取整个文件，再将文件分为行列表  
这种方法占内存  
fread = fhand.readlines()
